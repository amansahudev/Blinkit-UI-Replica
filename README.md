## 🚀 Features

* **Fully Responsive Design:** Utilizes modern CSS Flexbox to ensure a seamless and fluid experience across desktop, tablet, and mobile devices.
* **Interactive DOM Manipulation:** Features a dynamic "Add to Cart" counter (`- 1 +`) built with Vanilla JavaScript, mimicking real e-commerce functionality and state management.
* **Modern CSS Architectures:** Implements CSS variables (`:root`) for consistent theming, easy color updates, and scalable styling.
* **Clean & Optimized Code:** Follows the DRY (Don't Repeat Yourself) principle with clean class naming and structured component styling.
* **Horizontal Scrolling:** Native mobile-friendly horizontal scroll containers for product categories.

## 🛠️ Tech Stack

* **HTML5:** Semantic markup and structured layout.
* **CSS3:** Advanced Flexbox, pseudo-classes, hover transitions, and responsive media queries.
* **Vanilla JavaScript:** Event listeners and dynamic element creation for cart interactions.
